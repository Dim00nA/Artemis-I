## Стек используемых технологий

Python 3
Используется как основной язык программирования, благодаря простоте синтаксиса и большому количеству библиотек для анализа, логирования и визуализации.

Библиотека kRPC
Позволяет управлять ракетой в игре Kerbal Space Program в реальном времени.
Через неё реализованы: считывание телеметрии; управление тягой двигателя; автопилот; отделение ступеней;тповороты ракеты; постановка на орбиту.

Matplotlib
Используется для построения графиков:
– “Высота vs Время”
– “Скорость vs Время”
– “Апоцентр vs Время”
– “Перицентр vs Время”
– “Угол vs Время”

Pandas
Используется для записи и обработки данных полёта в формате CSV.

CSV-файл логирования, применяется для сохранения телеметрии и дальнейшего анализа траектории.
